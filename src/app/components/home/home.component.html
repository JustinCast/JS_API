Getting started

Para subir las propias funciones, primero debe iniciar sesión en la parte derecha de la pantalla.

Importar una librería específica

Si se desea importar una librería específica, basta solo con presionar cualquier librería en la sección de las funciones

Como puede ver, el script se genera en la parte de abajo cuando abre cada card

Importar varias librerías

Si se desean importar varias librerías, se debe hacer de la siguiente manera

Este script debe colocarle en el archivo index.html de su proyecto (o su archivo equivalente)
Al método importar se le debe enviar por parámetro, los identificadores de las funciones
